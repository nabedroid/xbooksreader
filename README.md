# 同人誌ビューワー

1万冊以上のローカル同人誌を管理・閲覧するためのWindowsネイティブデスクトップアプリケーション。

## 特徴

- **大規模コレクション対応**: 1万冊以上の同人誌を高速に管理
- **ZIP対応**: 解凍せずにZIPファイル内の画像を直接閲覧
- **メタデータ管理**: タイトル、シリーズ、作者、タグ、評価などを管理
- **しおり機能**: お気に入りのページにしおりを追加
- **タブ機能**: 複数の本を同時に開いて切り替え
- **検索・フィルター**: キーワード、シリーズ、作者、評価で検索
- **画像ビューワー**: ズーム、ページナビゲーション機能
- **バックアップ/エクスポート**: メタデータのJSON出力・データベースバックアップ
- **ポータブル**: インストール不要、単一EXEで動作

## 技術スタック

- **フレームワーク**: Electron 28+
- **UI**: React 18 + TypeScript
- **データベース**: SQLite
- **画像処理**: sharp
- **ZIP処理**: yauzl
- **状態管理**: Zustand
- **ビルドツール**: Vite

## 開発環境のセットアップ

### 前提条件

- Windows 10/11

### インストール

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build
```

## 使用方法

### 初回起動

1. アプリケーションを起動
2. 設定画面で同人誌フォルダを指定
3. スキャンを実行してライブラリを構築

### キーボードショートカット

| ショートカット | 機能 |
|--------------|------|
| `←` / `→` | 前/次のページ |
| `Shift + ←/→` | 10ページジャンプ |
| `Ctrl + ←/→` | 100ページジャンプ |
| `+` / `-` | ズームイン/アウト |
| `0` | ズームリセット |
| `F` | フルスクリーン |
| `B` | しおり追加 |
| `Ctrl + T` | 新しいタブ |
| `Ctrl + W` | タブを閉じる |
| `Ctrl + F` | 検索 |
